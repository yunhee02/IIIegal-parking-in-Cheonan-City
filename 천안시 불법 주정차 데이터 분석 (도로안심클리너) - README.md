# 🚗 천안시 불법 주정차 데이터 분석 (도로안심클리너)

## 📖 프로젝트 개요
천안시 불법 주정차 단속 데이터를 활용하여  
단속 패턴을 분석하고, 교통 혼잡 구간 및 운영 개선 방안을 도출한 프로젝트입니다.

---

## 🎯 분석 목적
- 불법 주정차 발생 패턴 분석
- 주요 혼잡 구간(Hotspot) 도출
- 데이터 기반 단속 효율 개선 방안 제시

---

## 🛠️ 사용 기술
- Python (Pandas)
- 데이터 전처리 및 정제
- 지오코딩 및 좌표 변환
- 데이터 시각화 (Tableau / 그래프 분석)

---

## 📊 주요 분석 내용
- 주소 데이터 기반 위도·경도 생성 (지오코딩)
- 위치 데이터 오류 및 불일치 문제 해결 (수작업 보정)
- 시간대별 / 지역별 단속 패턴 분석
- 공간 밀도 분석을 통한 혼잡 구간 도출
- 주차 인프라와 단속 데이터 간 상관 분석

---

## 📐 분석 방법
- API 기반 지오코딩 → 데이터 품질 문제 확인
- 수작업 데이터 보정 (약 1,000건)
- 시간대별 / 지역별 집계 분석
- 공간 밀도 기반 Hotspot 분석
- 상관계수 분석 (주차 인프라 vs 단속)

---

## 💡 결과 및 인사이트
- 단속은 특정 시간대(10시, 14~20시)에 집중 발생
- 주요 혼잡 구간 Top10이 전체 단속의 약 13% 차지
- 주차 인프라 부족과 단속 발생 간 양의 상관관계 확인 (0.31~0.42)
- 행정구역 기준 관리의 한계를 발견하고  
  → Hotspot 기반 운영 전략 필요성 제시
